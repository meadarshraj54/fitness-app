# Fitness App – Spring Boot REST API (Dockerized)

A Spring Boot backend application for managing fitness activities and user-specific recommendations. The project is built with a focus on security, clean architecture, and scalability, making it suitable for learning, development, and deployment.

All APIs are secured using JWT-based authentication and role-based access control (RBAC). The application uses MySQL for persistent storage and is fully Dockerized for consistent execution across environments.

---

## Features

- RESTful APIs for fitness activities and recommendations  
- JWT-based authentication with Spring Security  
- Role-Based Access Control (RBAC)  
- MySQL integration using Spring Data JPA  
- Input validation using Jakarta Validation  
- Dockerized for easy setup and deployment  

---

## Tech Stack

- Java  
- Spring Boot  
- Spring Security  
- JWT  
- Spring Data JPA  
- MySQL  
- Docker
